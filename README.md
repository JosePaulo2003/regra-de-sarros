# Fluxograma da Regra de Sarrus

Este projeto apresenta um fluxograma inteiramente em HTML e CSS que explica o cálculo do determinante de matrizes 3×3 utilizando a Regra de Sarrus. A proposta é oferecer um material didático, em português, com animações que ajudam a visualizar cada etapa do processo.

## Estrutura do projeto

- **index.html**: contém toda a estrutura do fluxograma, com os cartões de cada etapa, legendas e representações visuais das matrizes.
- **style.css**: agrega os estilos, paleta de cores, efeitos de transição e animações ativadas ao passar o mouse pelos cartões.

## Funcionalidades

- Destaque interativo das diagonais principais e secundárias, com repetição das colunas para ilustrar a regra.
- Cartões que se expandem ao hover para revelar detalhes das somas (SOMA1 e SOMA2) e do resultado final.
- Layout responsivo pensado para leitura confortável em diferentes tamanhos de tela.

## Como visualizar

1. Faça o download ou clone o repositório.
2. Abra o arquivo `index.html` diretamente no navegador de sua preferência.
3. Passe o mouse sobre os cartões para ver as animações que explicam cada passo.

## Personalização rápida

- Ajuste cores e fontes editando variáveis e gradientes no `style.css`.
- Substitua os textos dos cartões no `index.html` caso deseje adaptar para outra linguagem ou conteúdo.

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge ou Safari).
- Não há dependências externas ou scripts adicionais.

## Créditos

Projeto elaborado para fins educacionais, destacando de forma visual a Regra de Sarrus para matrizes 3×3.
